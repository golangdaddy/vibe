# Minecraft HTML5

A simple Minecraft-like game implemented using HTML5 and Three.js.

## Features

- 3D procedurally generated terrain
- First-person camera controls
- Basic block types (grass, dirt, stone)
- WASD movement and mouse look controls

## How to Run

1. Simply open the `index.html` file in a modern web browser
2. Click anywhere on the game window to start controlling the camera
3. Press Esc to release mouse control

## Controls

- W: Move forward
- S: Move backward
- A: Move left
- D: Move right
- Mouse: Look around
- Click: Enable mouse control
- Esc: Disable mouse control

## Technical Details

The game is built using:
- HTML5
- Three.js for 3D rendering
- Pure JavaScript for game logic

## Requirements

- A modern web browser with WebGL support
- No additional installation needed
