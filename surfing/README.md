# Top-Down Driving Game

A simple browser-based driving game inspired by classic top-down driving games like GTA 1.

## How to Play

1. Open `index.html` in a web browser
2. Use the following controls:
   - **Up Arrow** to accelerate
   - **Down Arrow** to brake/reverse
   - **Left Arrow** to turn left
   - **Right Arrow** to turn right
3. Drive around the city while avoiding buildings

## Game Features

- Top-down driving mechanics
- City layout with buildings and roads
- Car physics including acceleration and rotation
- Collision detection with buildings

## Technical Details

The game is built using:
- HTML5 Canvas for rendering
- Vanilla JavaScript for game logic
- CSS for basic styling
